# 🐍 貪食蛇遊戲 Snake Game

一個美觀的網頁版貪食蛇遊戲，支援電腦和手機遊玩！

## 🎮 立即遊玩

👉 **[點擊這裡開始遊戲](https://alexbbno1.github.io/Snake-game/)**

## ✨ 遊戲特色

| 功能 | 說明 |
|------|------|
| 🎨 霓虹風格設計 | 漂亮的漸層與發光效果 |
| 📱 響應式設計 | 支援電腦和手機遊玩 |
| 🏆 最高分紀錄 | 自動儲存在瀏覽器中 |
| ⚡ 漸進式難度 | 分數越高，速度越快 |
| 🔄 即時重玩 | 遊戲結束後可立即重新開始 |

## 🕹️ 操作方式

### 電腦
- `↑` `↓` `←` `→` 方向鍵控制
- `W` `A` `S` `D` 鍵控制
- `空白鍵` 開始遊戲

### 手機
- 使用螢幕上的方向按鈕

## 🛠️ 技術

- **HTML5 Canvas** - 遊戲渲染
- **CSS3** - 霓虹風格動畫效果
- **Vanilla JavaScript** - 遊戲邏輯
- **LocalStorage** - 最高分儲存
- **響應式設計** - 支援各種螢幕尺寸

## 🚀 本地運行

1. 下載或克隆此倉庫
2. 用瀏覽器開啟 `index.html`
3. 開始遊戲！

```bash
git clone https://github.com/AlexBBno1/Snake-game.git
cd Snake-game
# 用瀏覽器開啟 index.html
```
